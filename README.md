# iTask – Your Task Planner

A simple todo app built with React and Tailwind CSS. Add, edit, delete, and mark tasks as complete, with data persisted in local storage.

## Features
- Add new todos (minimum 4 characters)
- Edit and delete existing todos
- Mark todos as complete/incomplete
- Toggle visibility of finished todos
- Data saved locally via `localStorage`

## Tech Stack
- React
- Tailwind CSS
- react-icons
- uuid

## Getting Started

```bash
git clone https://github.com/amrit-kaur45/task-tracker.git
cd task-tracker
npm install
npm run dev
```

Open `http://localhost:5173` in your browser.
